Описание проекта: 
{
    Сайт, на котором будет храниться определенная база данных, с информацией поступившей от пользователя с 
    предварительной модерацией. 
    Шапка:
    {
        Приветствие пользователя, некоторое описание того, что это за сайт 
    }
    Необходимые элементы:
    {
        Инструменты для пользователя, с описанием их использования для создания новой статьи 
        Форум, на котором пользователи смогут обсуждать то, как правильно можно описать те или иные события или предмет
        Участие, для объединения пользователей для описания интересующих их статей совместно 
        Печать, экспорт с информацией о том, как можно экспортировать или создать книгу самостоятельно 
        Языки, которые пользователь может выбрать и корректировать информацию для конкретных стран на их языке 
        Связь между статьями, каждая статья должна иметь свой ID и если это слово встречается в другой статье
        должна появиться гиперссылка. 
    }
    Необходимые блоки:
    {
        Текущие события, описанные пользователями для того, чтобы люди знали о событиях в мире 
        Новые материалы - новые статьи от пользователей
        Совместная работа, как форум, для пользователей, которые ищут команду для написания статьи 
    }
    Главные кнопки:
    {
        Создать статью - для пользователей, которые хотят создать новую статью 
        Справка - с информацией о том, как это сделать 
        Поиск по статьям
    }
    Информация:
    {
        Расположение серверов
        Другие наши ресурсы
        Политика конфиденциальности
        Связь с нами 
        Статистика
    }
}
